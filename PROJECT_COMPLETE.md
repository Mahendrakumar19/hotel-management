## 🎓 HOW TO GET STARTED

### Step 1: Clone from GitHub
```bash
git clone https://github.com/Mahendrakumar19/hotel-management.git
cd hotel-managemnt
```

### Step 2: Run Setup
```bash
# Windows
setup.bat

# Linux/Mac
chmod +x setup.sh
./setup.sh
```

### Step 3: Start Services
```bash
# Terminal 1 - Backend
cd backend
npm run dev

# Terminal 2 - Frontend
cd frontend
npm run dev
```

### Step 4: Access Application
- Frontend: http://localhost:3000
- Backend API: http://localhost:5000/api
- Health Check: http://localhost:5000/health

### Step 5: Login
Default credentials:
- Admin: admin@hotel.com / admin@123
- Front Desk: frontdesk1@hotel.com / frontdesk@123
- F&B: fb@hotel.com / fb@123
