## 📋 OPERATIONAL WORKFLOWS SUPPORTED

### Guest Lifecycle
```
1. Reservation
   └─→ Arrival Notification

2. Check-In
   ├─→ Room Allocation
   ├─→ Registration
   └─→ Ledger Creation (Room Charges)

3. During Stay
   ├─→ F&B Billing (Optional)
   ├─→ Room Service Adjustments
   └─→ Real-time Ledger Updates

4. Checkout
   ├─→ Ledger Review
   ├─→ Payment Processing
   ├─→ Room Status Update (Dirty)
   └─→ Checkout Confirmation

5. Room Reset
   └─→ Housekeeping (Manual) → Room Status (Vacant)
```

### Room Management
```
Vacant → Reserved
            ↓
         Check-In → Occupied
                      ↓
                   Check-Out → Dirty
                                  ↓
                            (Cleaning) → Vacant
```

### F&B Billing
```
Bill Created (Open)
   ↓
Add Items (Food, Beverage, Service)
   ↓
Calculate Total
   ↓
Select Settlement Mode (Cash, Card, UPI, Room Charge)
   ↓
Settle Bill (Mark as Settled)
   ↓
Update Guest Ledger & Reports
```
